The Logic of Sonic Mapping

The Premise

Sonic Stories is a companion art project of `1000 Songs`. If `1000 Songs` maps the chronology of personal music discovery across a century, `Sonic Stories` maps the spatial landscape. Songs aren't just listened to in isolation; they are bound to damp basements, concrete amphitheaters, starry hills, and long highway drives.

JSON Schema Details

Curations connect dynamically to JSON inputs. Standard objects contain:
- id: string
- title: string
- author: string
- year: number
- locationName: string
- lat: number (latitude)
- lng: number (longitude)
- artist: string
- track: string
- content: string
- tags: array of strings (e.g. ["gig", "punk", "london"])
- overrideUrl: string (optional iTunes ID or direct URL)

Direct Submissions

By using the Submit portal, your entries are automatically delivered to the registry's creator via secure channels. These stories are verified manually for formatting accuracy and added to the core spatial database.
